## Overview
This project implements an AI-based Intrusion Detection System using Machine Learning techniques to detect malicious network traffic.

## Features
- Data preprocessing and feature engineering
- Random Forest & XGBoost models
- Feature importance analysis
- Flask-based web dashboard
- UNSW-NB15 dataset support

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Flask
- Matplotlib

## Dataset
- UNSW-NB15 Intrusion Detection Dataset
- Available at: [UNSW-NB15 Dataset](https://www.unsw.adfa.edu.au/unsw-canberra-cyber/cybersecurity/ADFA-NB15-Datasets/)
